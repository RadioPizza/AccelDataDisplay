Detailed Description

Таблица шрифтов для 5x7 символов, хранимая во FLASH.

Данный файл содержит массив шрифта для отображения символов размером 5x7. Каждый символ занимает 5 байт (столбцов), где каждый байт описывает 7 вертикальных пикселей (бит).

Таблица шрифта 5x7.
