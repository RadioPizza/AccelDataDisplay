Function Documentation

◆ I2C_Init()

uint8_t I2C_Init ( I2C_Mode_t mode)

Инициализация интерфейса I2C.

Выполняет настройку периферийного модуля I2C для работы в одном из двух режимов:

  • Стандартный режим (Standard Mode) с частотой 100 кГц
  • Быстрый режим (Fast Mode) с частотой 400 кГц
Parameters
[in]modeВыбор режима работы I2C:
  • I2C_STANDARD_MODE: Стандартный режим (100 кГц)
  • I2C_FAST_MODE: Быстрый режим (400 кГц)
Returns
Результат инициализации
Return values
0Инициализация выполнена успешно
1Ошибка при инициализации
