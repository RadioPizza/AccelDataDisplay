Detailed Description

Реализация функций для работы с OLED дисплеем 128x64 (SSD1306) по протоколу I2C.

Очищает содержимое дисплея

Заполняет весь дисплей черным цветом (гасит все пиксели), подготавливая его к рисованию новой информации.

Отображает битовое изображение на дисплее

Рисует изображение на дисплее по заданным координатам. Изображение должно быть закодировано в массиве bitmap.

Parameters
[in]xНачальная X-координата (колонка) от 0 до ширины дисплея
[in]yНачальная Y-координата (страница) от 0 до количества страниц дисплея
[in]bitmapУказатель на массив байтов, содержащий битовое изображение
[in]widthШирина изображения в пикселях
[in]heightВысота изображения в пикселях
Note
Формат изображения: вертикальная ориентация, 1 бит на пиксель.

< Высота в байтах (количество страниц по вертикали).

< Проход по страницам.

< Проход по пикселям по ширине.

Установка курсора в нужное место.

< y / 8 — определение страницы.

◆ SSD1306_Init()

void SSD1306_Init ( void )

Инициализирует дисплей SSD1306.

Настраивает основные параметры дисплея, включая разрешение, режим работы и интерфейс связи. Необходима для подготовки дисплея к приему данных.

Warning
Функция не проверяет корректность инициализации I2C. Убедитесь, что шина I2C настроена правильно перед вызовом.

uint8_t page )

Устанавливает позицию курсора

Задает текущую позицию курсора для последующей отрисовки символов или графических объектов.

Parameters
[in]columnКолонка (X-координата) от 0 до 127
[in]pageСтраница (Y-координата) от 0 до 7 (для дисплея 128x64)
Note
Координаты начинаются с нуля. Убедитесь, что указанные значения находятся в допустимых пределах.
